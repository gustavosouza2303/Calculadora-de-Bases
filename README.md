# 🔢 Base Converter & ASCII Machine

Projeto desenvolvido para conversão de bases numéricas e visualização de códigos de caracteres (ASCII/Unicode)

---

### 🔁 Conversão de Bases

* Decimal ↔ Binário
* Decimal ↔ Octal
* Decimal ↔ Hexadecimal

Permite inserir um valor em qualquer base e obter automaticamente as demais.

---

### 🔤 ASCII / Unicode Machine

Converte um caractere e exibe:

* 🔢 Valor Decimal (ASCII)
* 💻 Binário
* 🧮 Hexadecimal
* 🌐 Unicode (U+XXXX)
* 📦 UTF-8 (em bytes hexadecimais)
* 🧾 UTF-16

---

## 🛠️ Linguagens utilizadas

* HTML
* CSS
* JavaScript

---

## 📚 Conceitos aplicados

Este projeto explora conceitos como:

* Sistemas de numeração (base 2, 8, 10, 16)
* Conversão entre bases
* Codificação de caracteres
* Unicode vs UTF-8 vs UTF-16
